Snackbars

Snackbar are simple popup message boxes. Use the class .snackbarto create a hidden snackbar which can be poped by calling the id using snackbar() function.

Note that you need to include beauter.js at the end for the dynamic toggle action of accordion button. See how here.

Example

This is a simple one.

<div id="sn1" class="snackbar">
  Hi there!!!
</div>
<button onclick="showsnackbar('sn1')">Here</button>
Hi there!!!
