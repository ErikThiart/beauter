Pagination

Beauter adds some inbuilt CSS utilities to make it easer to use.

base 
pagination

As disabled and active page numbers are not hyper linked, hence use <span>.

Example 

<div class='pagination'>
	<span class = '_disabled'>« previous</span>
	<span class = '_active _round'>1</span>
	<a href = '#page1'>2</a>
	<a href = '#page2'>next »</a>
</div>
